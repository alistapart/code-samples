“Behind the scene” content

We can take advantage of the way the elements are styled to insert content into the wrapper, but note that IE5 increases the height of the video to make room for that text, creating black stripes at the top and bottom of the video).

Description of the video embeded in this document

This short clip is about YouTube widescreen formatting. It shows the two main formats (16:9, 4:3) and also explains the best way to create a Flash movie according to the new widescreen format.

CSS

#containingBlock {
  width:50%;
}
.videoWrapper {
  position: relative;
  padding-bottom:56.25%;
  padding-top:25px;
  height:0;
  overflow:hidden;
}
.videoWrapper div,
.videoWrapper object {
  position:absolute;
  top:0;
  left:0;
  width: 100%;
  height:100%;
}
