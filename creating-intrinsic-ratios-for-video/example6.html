The single-object method

The single-object method encodes the ampersands in the URLs using a solution inspired by the “nested-objects” method (but reduced to one single object).

CSS

#containingBlock {
  width:50%;
}
.videoWrapper {
  position: relative;
  padding-bottom:56.25%;
  padding-top:25px;
  height:0;
}
* html .videoWrapper {
  margin-bottom: 45px;
  margin-bot\tom: 0;
}
.videoWrapper div,
.videoWrapper object {
  position:absolute;
  top:0;
  left:0;
  width: 100%;
  height:100%;
}
